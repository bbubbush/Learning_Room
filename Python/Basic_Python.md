## if문

- **if문의 기본적인 포맷** 

  - if문의 다음 줄은 반드시 ***들여쓰기***

  ```python
  # 올바른 사용법
  if 조건식 :
      print()
      print()
      print()
      
  # 잘못된 사용법 - 오류발생
  if 조건식 :
      print()
    print()
         print()
  ```

  - if문의 블럭관계 - 들여쓰기의 중요성

  ```python
  if True:
      print('조건1')
      if False:
          print('조건2')
      print('조건3')
  #조건3의 명령어는 첫 번째 if문의 소속
  @실행결과 : 조건1
      	   조건3
  ```

  > 조건 :  < , > , <= , >=, ==, !=

  > boolean : and , or , not 

- **if-else / elif기본적인 포맷**

  ```python
  if 조건식1:   
      code
  elif 조건식2:   
      code
  else:   
      code
  ```
